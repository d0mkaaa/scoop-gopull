# scoop-gopull

Scoop bucket for [gopull](https://github.com/d0mkaaa/gopull).

## Install

```powershell
scoop bucket add d0mkaaa https://github.com/d0mkaaa/scoop-gopull
scoop install gopull
```

## Upgrade

```powershell
scoop update gopull
```

---

The manifest `gopull.json` is auto-generated by GoReleaser on each release.
Do not edit it by hand.
